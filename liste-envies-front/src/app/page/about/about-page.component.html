Une liste d'envie UNIQUE en son genre,

Voici LA liste d'envies qui vous permet de partager vos envies de cadeaux avec vos proches. Ainsi, ces derniers peuvent indiquer quel(s) cadeau(x) ils souhaitent vous offrir, se mettre d'accord entre eux,(en évitant les doublons!)sans que vous en soyez informé. Vous aurez la surprise le jour j !

Voir les listes de démo
Créer votre liste d'envies en ligne

Vous pouvez créer simplement une liste de tous les cadeaux que vous souhaitez avoir pour un événement particulier (Noël, Anniversaire, Mariage, Naissance ...)

Partagez votre liste de manière privée

Partagez votre liste seulement a certaines personne de manière privée ou laissez l'accès à tout le monde.
Ainsi toutes les personnes à qui vous partagez votre liste pourront se coordonnées pour les cadeaux.

Gardez la surprise pour le jour J

Vous ne verrez pas qui vous offre un cadeau. Vous garderez la surprise lorsque vous recevrez le cadeau. A l'inverse toutes les personnes à qui vous partagerez la liste verront ce qui est déjà offert pour éviter les doublons.
